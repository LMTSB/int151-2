*** Events ***
-----
- Design Pattern is a reusable solution for a common problem that occurs in software design.
- Event Propagation describes how events (clicks ,ketpressed and etc.) travel through DOM(Document Object Model) when something happens on webpage.
- Event Handlers is a function that executes when that event occurs.
- Event Handlers + Event Propagation(Capturing and Bubbling) in DOM 
- The Browsers trigger the events and move it through 3 phases 1.Capturing(Top -> Target) 2.Targetting 3.Bubbling(Target -> Top)
- PreventDefault function In every browser often comes with built-in
